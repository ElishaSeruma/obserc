Obserc Beta Release Notes

Every update brings us closer to launch. Below is a detailed breakdown of changes, improvements, and known issues for each Obserc beta release.

v0.8.2 Beta (UI Preview)

Release Date: August 10, 2025

Type: UI Build

Highlights

  • Circle management added directly to the dashboard for seamless access.
  • Manual escalation button integrated into the chat interface.
  • Detection engine now processes 12% faster under low signal conditions.
  • Updated iconography and micro-animations for improved feedback loops.

Fixes

  • Resolved duplicate emergency alerts in low-network mode.
  • Fixed UI freeze when Circle exceeded 10 members.

Known Issues

  • Dark mode resets after app restart.
  • On some Android tablets, chat messages may display misaligned.
Download v0.8.2

v0.7.9 Beta (Headless)

Release Date: August 02, 2025

Type: Headless Build

Highlights

  • Expanded voice detection range for verbal distress signals.
  • Tuned AI to reduce false positives in crowded environments.
  • Added background performance logging for analytics.

Known Issues

  • Occasional missed physical motion triggers on low-tier devices.
  • Inconsistent feedback tones when escalation is triggered.
